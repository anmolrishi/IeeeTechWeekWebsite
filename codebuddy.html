Codebuddy

Teams of 2 will compete in a time bounded coding contest in which one member will have to write code first for half duration and the other one will have to complete the code. The members will not be allowed to communicate at any point of time during the contest.

Rules

1)No communication through any means will be allowed
2) Putting comments in the code will be prohibited
3) Variables can have a max length of 15 characters. (Can be changed)
4) Method names, class names will also have limit on the number of characters
5) Any variable, method, import declared should be used in the code, i.e. no unnecessary variables etc allowed


Judging criteria


1) Two evaluations be done- First after half time. Any defaulters will be eliminated Second after full time.
2) Winner will be selected on the basis of correctness of the code/completeness. In case of a tie the code closest to the solution will be the winner
